Class EntityClient

Convenience wrapper around the AWS SDK DynamoDBClient and DynamoDBDocument classes with enhanced batch processing.

Remarks

This class provides a number of enhanced methods. For everything else, both the DynamoDBClient and DynamoDBDocument instances are exposed for direct access on the client and doc properties, respectively.
